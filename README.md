BME msc első féléves önlab: Futball statisztikai alkalmazás fejlesztése Android platformra

Technológiák:
  - MVVM architektúra
  - API adatok használata
  - Firebase authentikáció
  - Jetpack Compose
  - Hilt (Dependency Injection)
  - Retrofit API hívásokhoz
